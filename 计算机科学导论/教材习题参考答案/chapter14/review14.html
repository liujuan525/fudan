一 选择题     二 简答题

1.研究、设计和应用智能机器或智能系统来模拟人类智能活动的能力,以延伸人类智能的科学是()

A.人工智能       B.遗传算法       C.机器学习       D.模式识别
参考答案：

A


2.模拟人类的听觉、视觉等感觉功能,对声音、图像、景物、文字等进行识别的方法是( )

A.专家系统       B.遗传算法       C.神经网络       D.模式识别
参考答案：

D


3.模拟人类专家对问题的求解过程,解决那些只有专家才能解决的复杂问题的系统是( )

A.证券管理决策系统       B.企业短期规划支持系统
C.专家系统 D.产品推销、定价决策系统
参考答案：

C


4.人类对客观世界及其内部运行规律的认识与经验的总和是( )

A.智能       B.知识       C.识别       D.专家
参考答案：

B


5.使用if-then 规则求解问题属于( )

A.消解演绎推理
B.规则演绎推理
C.与/或形演绎推理
D.产生式系统
参考答案：

B


6.用模糊性知识进行的一种不确定性推理的是( )

A.模糊推理       B.不确定性推理       C.非单调推理       D.时序推理
参考答案：

C


7.从开始状态开始搜索,然后是开始状态的一个后继状态(即通过一次状态转移可以到达的状态),接着是该状态 的一个后继状态,如此等等,直至到达目标状态,这种搜索技术是( )

A.广度优先搜索法
B.等代价搜索法
C.深度优先搜索法
D.其他搜索法
参考答案：

C

二 简答题

题号题目
1什么是人工智能？
参考答案：

人工智能AI(Artificial Intelligence)，又称为机器智能MI(Machine Intelligence)，是研究、设计和应用智能机器或智能系统，用来模拟人类智能活动的能力，以延伸人类智能的科学。它是一门综合了计算机科学、生理学、控制论、信息论、神经生理学、语言学、哲学的交叉学科。
