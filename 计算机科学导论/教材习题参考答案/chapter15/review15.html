第15章课后习题





一 选择题     二 简答题

1.当讨论一种行为是否正确时,这个问题应该属于的研究领域是()

A.标准       B.道德       C.原则       D.以上都不是
参考答案：

B


2.以下行为中,( )是不道德的

A.避免伤害别人       B.尊重秘密
C.讲真话       D.复制别人购买的版权软件
参考答案：

D


3.当使用别人的知识产权时,应该( )

A.注明作者的名字       B.不泄露所发现的技术秘密
C.不复制软件 D.以上都是
参考答案：

D


4.一个人可以合法复制的软件是( )

A.免费软件       B.共享软件
C.A和B       D.任何未经允许的复制都是非法的
参考答案：

C


5.以下属于软件盗版的行为包括( )

A.盗窃计算机磁盘
B.复制别人购买的版权软件
C.程序员从其他程序中拷贝代码
D.盗窃计算机芯片
参考答案：

B


6.知识财产包括( )

A.商业机密       B.版权       C.专利       D.以上都是
参考答案：

D


7.黑客行为包括( )

A.闯入计算机系统的犯罪行为
B.测试计算机系统的极限
C.非法复制软件
D.A和B
参考答案：

D


8.实施计算机犯罪的人员包括( )

A.内部工作人员
B.外部的授权访问者
C.黑客
D.未经授权的解密者
参考答案：

CD
